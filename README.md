# Agent Framework Learning Lab

This repository is a personal learning project exploring **agent framework patterns**, 
with a focus on **LangGraph-style orchestration** and how those ideas map to 
**Microsoft Agent Framework concepts**.

## Goals
- Understand agent orchestration and state graphs
- Learn tool-based data access patterns
- Explore observability-style agents (logs, errors, summaries)
- Compare LangGraph concepts with Microsoft Agent Framework

## What This Is
- A sanitized, non-production learning project
- Uses mock / example schemas
- Focuses on architecture, not business logic

## What This Is NOT
- Company code
- Production-ready software
- Connected to real systems

## Tech
- Python
- LangGraph
- Tool calling
- Agent state graphs
